# pomodoro-timer
My first Pomodoro timer was built with HTML, CSS, and JavaScript. Fitted with buttons labeled "Start", "Pause", and "Reset". I created this website as a submission to Hack Club Stasis. My end goal is to hopefully earn a Flipper Zero, which I can use as a convenience factor for so many different things! So in other words, this project is created for personal use, public use, and to earn "Bits".
